O ficheiro P.mat contém uma matriz com pontos (x,y,z) de uma seta.
Este ficheiro pode ser carregado em matlab com o seguinte comando:
	load P
que carrega na variável P a matriz guardada no ficheiro P.mat

Este objeto representa, com 70 pontos, a figura de uma seta paralela
ao plano XY mas à cota 30 no plano Z. 

Este objeto pode ser usado em alternativa à circunferência
de pontos usada no exercício 4.
